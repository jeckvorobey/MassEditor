## Purpose

Зафиксировать требования к воспроизводимой и безопасной сборке релизного архива плагина MassEditor для Webasyst Store.

## Requirements

### Requirement: Источник и имя релизного архива
Система сборки MUST (ДОЛЖНА) формировать `dist/masseditor-1.1.0.tar.gz` только из актуального production-дерева `wa-apps/shop/plugins/masseditor/`, а версия в имени ДОЛЖНА совпадать с трехчастной версией из `lib/config/plugin.php`.

#### Scenario: Сборка текущей версии
- **WHEN** `plugin.php` содержит числовой `vendor` и версию `1.1.0`
- **THEN** результат ДОЛЖЕН быть создан как `dist/masseditor-1.1.0.tar.gz`
- **AND** распакованное содержимое ДОЛЖНО соответствовать текущему production-дереву плагина

#### Scenario: Некорректная версия или vendor
- **WHEN** версия не состоит из трех чисел, не равна ожидаемому релизу или `vendor` не является числовым
- **THEN** сборка ДОЛЖНА завершиться ошибкой до замены существующего архива

### Requirement: Формат архива Webasyst Store
Релизный артефакт MUST (ДОЛЖЕН) быть корректным `tar.gz` архивом с единственной корневой директорией `masseditor/` и безопасными относительными путями.

#### Scenario: Проверка корневой структуры
- **WHEN** читается manifest собранного архива
- **THEN** каждый путь ДОЛЖЕН начинаться с `masseditor/`
- **AND** архив НЕ ДОЛЖЕН содержать второй верхнеуровневый файл или каталог

#### Scenario: Проверка безопасных путей
- **WHEN** проверяется manifest архива
- **THEN** он НЕ ДОЛЖЕН содержать абсолютные пути, переходы `..`, символические ссылки за пределы корня или специальные файлы

### Requirement: Только production-файлы плагина
Архив MUST (ДОЛЖЕН) содержать все файлы production-дерева и НЕ ДОЛЖЕН содержать служебные материалы репозитория, временные файлы, debug-код или ненужные пустые директории.

#### Scenario: Полный production-manifest
- **WHEN** кандидат распакован во временную директорию
- **THEN** рекурсивное сравнение с `wa-apps/shop/plugins/masseditor/` ДОЛЖНО завершиться без различий

#### Scenario: Исключение материалов разработки
- **WHEN** проверяется manifest кандидата
- **THEN** в нем НЕ ДОЛЖНО быть `docs/`, `tests/`, `openspec/`, `docker/`, `.git/`, `.idea/`, `.codex/`, `AGENTS.md`, логов, временных файлов или резервных копий редакторов

#### Scenario: Отсутствие debug-артефактов
- **WHEN** production-файлы проходят preflight-поиск
- **THEN** release paths НЕ ДОЛЖНЫ содержать `var_dump`, `print_r`, `die`, `exit` или `console.log`, оставленные для отладки

### Requirement: Защищенные директории и публикационные метаданные
Архив MUST (ДОЛЖЕН) сохранять Webasyst-структуру плагина, обязательные защитные файлы и декларацию совместимости Shop-Script.

#### Scenario: Проверка защищенных директорий
- **WHEN** проверяются `masseditor/lib/`, `masseditor/templates/` и `masseditor/locale/` в распакованном архиве
- **THEN** каждая корневая защищенная директория ДОЛЖНА содержать `.htaccess` с единственной директивой `Deny from all`

#### Scenario: Проверка конфигурации Shop-Script
- **WHEN** проверяются файлы `lib/config/plugin.php` и `lib/config/shop_support.json`
- **THEN** конфигурация ДОЛЖНА содержать ID-контекст `masseditor`, числовой `vendor`, версию `1.1.0` и явную декларацию поддержки или отсутствия поддержки PREMIUM-возможностей

### Requirement: Актуальные локализации
Релизный архив MUST (ДОЛЖЕН) содержать проверенные пары `.po` и `.mo` для `ru_RU` и `en_US`, собранные из текущих исходников переводов до упаковки.

#### Scenario: Компиляция обеих локалей
- **WHEN** выполняется релизная сборка
- **THEN** каждый `shop_masseditor.po` ДОЛЖЕН успешно пройти gettext-проверку и сформировать соответствующий `shop_masseditor.mo`

#### Scenario: Инструмент локализации недоступен
- **WHEN** проверенный gettext-совместимый компилятор недоступен или сообщает ошибку
- **THEN** сборка ДОЛЖНА остановиться и НЕ ДОЛЖНА заменять существующий архив

### Requirement: Релизный quality gate
Финальный архив MUST (ДОЛЖЕН) заменять предыдущий файл только после успешных runtime-, syntax-, structural- и content-проверок.

#### Scenario: Успешная публикационная сборка
- **WHEN** PHP/JS-тесты, syntax checks, gzip/tar-проверки, manifest-проверки и сравнение распакованного кандидата завершились успешно
- **THEN** кандидат ДОЛЖЕН атомарно заменить `dist/masseditor-1.1.0.tar.gz`
- **AND** итоговый отчет ДОЛЖЕН содержать путь, размер и SHA-256 финального файла

#### Scenario: Ошибка до финальной замены
- **WHEN** любая обязательная проверка кандидата завершается ошибкой
- **THEN** существующий `dist/masseditor-1.1.0.tar.gz` НЕ ДОЛЖЕН быть заменен
- **AND** неуспешный кандидат ДОЛЖЕН быть удален из временного расположения
