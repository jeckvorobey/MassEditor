## Purpose

Зафиксировать мобильное поведение выбора всех товаров текущей страницы в backend UI MassEditor.

## Requirements

### Requirement: Мобильный чекбокс «Выбрать все на странице»
На мобильных устройствах (≤1024px) пользователь MUST иметь доступ к чекбоксу «Выбрать все на странице» в зоне действий таблицы товаров, визуально и функционально аналогичному десктопному чекбоксу в `thead`.

#### Scenario: Отображение на мобильном
- **WHEN** пользователь открывает вкладку товаров на устройстве с шириной ≤1024px
- **THEN** в зоне `.masseditor-table-card__actions` ДОЛЖЕН отображаться чекбокс с лейблом, использующим i18n-ключ `select_all_page`

#### Scenario: Скрытие на десктопе
- **WHEN** пользователь открывает вкладку товаров на устройстве с шириной >1024px
- **THEN** мобильный чекбокс «Выбрать все на странице» ДОЛЖЕН быть скрыт через CSS

#### Scenario: Выбор всех товаров на странице
- **WHEN** пользователь отмечает мобильный чекбокс «Выбрать все на странице»
- **THEN** все чекбоксы товаров на текущей странице ДОЛЖНЫ быть отмечены, а ID товаров ДОЛЖНЫ быть сохранены в `localStorage`

#### Scenario: Снятие выбора
- **WHEN** пользователь снимает отметку с мобильного чекбокса
- **THEN** все чекбоксы товаров на текущей странице ДОЛЖНЫ быть сняты, а ID товаров ДОЛЖНЫ быть удалены из `localStorage`

#### Scenario: Indeterminate состояние
- **WHEN** отмечена часть товаров на странице
- **THEN** мобильный чекбокс ДОЛЖЕН отображать промежуточное состояние (indeterminate)

#### Scenario: Синхронизация с десктопным чекбоксом
- **WHEN** пользователь использует десктопный чекбокс в `thead` (на широком экране) или мобильный чекбокс
- **THEN** оба элемента ДОЛЖНЫ отражать одно и то же состояние при смене breakpoint

### Requirement: i18n мобильного чекбокса
Мобильный чекбокс «Выбрать все на странице» MUST использовать существующий i18n-ключ `select_all_page` для `aria-label` и видимого текста лейбла.

#### Scenario: Русский язык
- **WHEN** `interface_language` равно `ru_RU`
- **THEN** лейбл мобильного чекбокса ДОЛЖЕН отображать текст из ключа `select_all_page` русского словаря

#### Scenario: Английский язык
- **WHEN** `interface_language` равно `en_US`
- **THEN** лейбл мобильного чекбокса ДОЛЖЕН отображать текст из ключа `select_all_page` английского словаря
