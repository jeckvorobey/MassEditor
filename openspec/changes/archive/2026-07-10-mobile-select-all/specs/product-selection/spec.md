## MODIFIED Requirements

### Requirement: Выбор всех товаров по фильтру
Выбор всех найденных товаров MUST выполняться серверной повторной выборкой ID по текущим фильтрам с учетом лимита операции. На мобильных устройствах (≤1024px) пользователь также ДОЛЖЕН иметь доступ к функции «Выбрать все на странице» через отдельный чекбокс в зоне действий таблицы, аналогичный десктопному чекбоксу в `thead`.

#### Scenario: Filter selection
- **WHEN** пользователь выбирает все товары по текущему фильтру
- **THEN** `getIdsByFilters()` MUST вернуть distinct ID в серверном порядке и ограничить выборку лимитом операции

#### Scenario: Мобильный выбор всех на странице
- **WHEN** пользователь на мобильном устройстве (≤1024px) отмечает чекбокс «Выбрать все на странице»
- **THEN** JS MUST отметить все видимые чекбоксы товаров на текущей странице и сохранить их ID в `localStorage`, аналогично десктопному поведению `data-role="select-all"`
