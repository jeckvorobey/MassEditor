## ADDED Requirements

### Requirement: Читаемая типографика backend UI
Backend UI MassEditor MUST использовать согласованную типографическую шкалу, в которой основной текст, табличные данные, подписи элементов и интерактивные controls остаются комфортно читаемыми на desktop и mobile.

#### Scenario: Таблицы и панели на desktop
- **WHEN** пользователь открывает вкладку `products`, `log` или `settings` на desktop-экране
- **THEN** основной текст панелей и таблиц MUST быть визуально читаемым без опоры на `10–12px` как основной размер для значений и действий

#### Scenario: Мобильные карточки и controls
- **WHEN** пользователь открывает backend UI на mobile-ширине
- **THEN** текст в карточках товаров, логе, кнопках, input и select MUST быть увеличен относительно desktop microcopy и MUST оставаться удобным для чтения и нажатия

#### Scenario: Scope без изменения пользовательского потока
- **WHEN** внедряется улучшение типографики
- **THEN** текущая структура вкладок, таблиц, карточек, confirm modal и локализованных текстов MUST сохраняться без изменения backend-логики
