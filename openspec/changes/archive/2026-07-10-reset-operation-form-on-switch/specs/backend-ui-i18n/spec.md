## MODIFIED Requirements

### Requirement: JavaScript без зависимостей
`js/masseditor.js` ДОЛЖЕН (MUST) оставаться простым dependency-free JavaScript, который управляет выбором товаров, localStorage, переключением операций, клиентской валидацией, confirm modal, toast-уведомлениями и подсказками поиска. При фактическом переключении на другую операцию JS ДОЛЖЕН (MUST) сбрасывать значения controls формы операций; выбор товаров и фильтры ДОЛЖНЫ сохраняться.

#### Scenario: Выбор товаров
- **WHEN** пользователь отмечает товары на странице
- **THEN** JS ДОЛЖЕН сохранить ID в `localStorage` с ключом, привязанным к plugin ID, и восстановить выбор при перезагрузке

#### Scenario: Смена операции
- **WHEN** пользователь выбрал другую операцию в списке операций
- **THEN** text/textarea предыдущей формы ДОЛЖНЫ очиститься, select ДОЛЖНЫ вернуться к первому варианту, а выбор товаров и фильтры ДОЛЖНЫ сохраниться

#### Scenario: Повторный выбор операции
- **WHEN** пользователь повторно выбирает уже активную операцию
- **THEN** заполненные значения этой операции ДОЛЖНЫ сохраниться

#### Scenario: Подтверждение операции
- **WHEN** пользователь открывает confirm modal
- **THEN** JS ДОЛЖЕН проверить обязательные поля текущей операции, заполнить summary и перед submit добавить hidden product IDs и `confirm_apply=1`
