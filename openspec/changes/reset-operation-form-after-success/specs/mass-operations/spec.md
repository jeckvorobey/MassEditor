## ADDED Requirements

### Requirement: Сброс формы после успешного применения
После успешного подтверждённого применения любой поддерживаемой массовой операции backend action MUST передать шаблону дефолтные параметры формы, очистить выбранные товары и открыть вкладку товаров. Права доступа, CSRF, серверная валидация, транзакционное применение и запись журнала ДОЛЖНЫ сохраниться без изменения.

#### Scenario: Успешная операция
- **WHEN** администратор подтверждает валидную массовую операцию и сервис завершает её успешно
- **THEN** backend action ДОЛЖЕН показать сообщение об успехе и дефолтные параметры формы вместо применённых значений

#### Scenario: Ошибка операции
- **WHEN** массовая операция завершается ошибкой валидации или выполнения
- **THEN** backend action ДОЛЖЕН сохранить введённые параметры и выбранные товары для исправления, не показывая ложный успешный результат
