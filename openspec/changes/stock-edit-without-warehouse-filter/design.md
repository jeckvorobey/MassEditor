## Approach

Дизайн будет дополнен по результатам исследования. На текущем этапе:

### Предварительный план

**Фильтр в UI**: при `stock_id = 0` показывать select с вариантами:
- Все товары (текущее поведение по умолчанию)
- Только без складского учёта
- Только со складским учётом

**Backend-логика**: перед `applyProductStockOperation()` — проверять `usesWarehouseStockAccounting($skus)` и сравнивать с значением фильтра. Если товар не попадает под фильтр — пропускать с записью в лог.

**Вопросы для исследования:**
1. Что происходит в Shop-Script, если обновить `sku.count` у товара со складским учётом?
2. Нужно ли синхронизировать `shop_product_stocks` при изменении `sku.count`?
3. Есть ли в Shop-Script API метод для корректного обновления остатков без указания склада?
