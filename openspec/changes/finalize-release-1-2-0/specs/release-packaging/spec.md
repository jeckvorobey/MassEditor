## MODIFIED Requirements

### Requirement: Источник и имя релизного архива
Система сборки MUST (ДОЛЖНА) формировать `dist/masseditor-1.2.0.tar.gz` только из актуального production-дерева `wa-apps/shop/plugins/masseditor/`, а версия в имени ДОЛЖНА совпадать с трехчастной версией из `lib/config/plugin.php`.

#### Scenario: Сборка текущей версии
- **WHEN** `plugin.php` содержит числовой `vendor` и версию `1.2.0`
- **THEN** результат ДОЛЖЕН быть создан как `dist/masseditor-1.2.0.tar.gz`
- **AND** распакованное содержимое ДОЛЖНО соответствовать текущему production-дереву плагина без различий

#### Scenario: Некорректная версия или vendor
- **WHEN** версия не состоит из трех чисел, не равна ожидаемому релизу или `vendor` не является числовым
- **THEN** сборка ДОЛЖНА завершиться ошибкой до замены существующего архива

### Requirement: Защищенные директории и публикационные метаданные
Архив MUST (ДОЛЖЕН) сохранять Webasyst-структуру плагина, обязательные защитные файлы и декларацию совместимости Shop-Script.

#### Scenario: Проверка защищенных директорий
- **WHEN** проверяются `masseditor/lib/`, `masseditor/templates/` и `masseditor/locale/` в распакованном архиве
- **THEN** каждая корневая защищенная директория ДОЛЖНА содержать `.htaccess` с единственной директивой `Deny from all`

#### Scenario: Проверка конфигурации Shop-Script
- **WHEN** проверяются файлы `lib/config/plugin.php` и `lib/config/shop_support.json`
- **THEN** конфигурация ДОЛЖНА содержать ID-контекст `masseditor`, числовой `vendor`, версию `1.2.0` и явную декларацию поддержки или отсутствия поддержки PREMIUM-возможностей

### Requirement: Релизный quality gate
Финальный архив MUST (ДОЛЖЕН) заменять предыдущий файл только после успешных runtime-, syntax-, structural- и content-проверок.

#### Scenario: Успешная публикационная сборка
- **WHEN** PHP/JS-тесты, syntax checks, gettext-проверки, gzip/tar-проверки, manifest-проверки и сравнение распакованного кандидата завершились успешно
- **THEN** кандидат ДОЛЖЕН атомарно заменить `dist/masseditor-1.2.0.tar.gz`
- **AND** итоговый отчет ДОЛЖЕН содержать путь, размер, число файлов и SHA-256 финального файла

#### Scenario: Ошибка до финальной замены
- **WHEN** любая обязательная проверка кандидата завершается ошибкой
- **THEN** существующий `dist/masseditor-1.2.0.tar.gz` НЕ ДОЛЖЕН быть заменен
- **AND** неуспешный кандидат ДОЛЖЕН оставаться только во временном расположении и НЕ ДОЛЖЕН считаться релизным артефактом
