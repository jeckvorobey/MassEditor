# Changelog

История изменений опубликованных версий этого продукта Webasyst.

## 1.0.0 — 2026-05-01

```html
<p><strong>Версия 1.0.0 — 1 мая 2026</strong></p>
<ul>
  <li>Выпущена первая версия плагина с массовыми операциями для цен, compare price, видимости, доступности, описаний, тегов и URL.</li>
  <li>Добавлены фильтрация товаров, пагинация и сохранение выбранных товаров при переходе между страницами списка.</li>
  <li>Добавлены подтверждение операций, журнал изменений и интерфейс на русском и английском языках.</li>
</ul>
```

## 1.1.0 — 2026-07-10

```html
<p><strong>Версия 1.1.0 — 10 июля 2026</strong></p>
<ul>
  <li>Добавлены массовое изменение остатков, фильтрация товаров по складу и отображение остатка выбранного склада.</li>
  <li>Расширено массовое редактирование характеристик: поддержаны логические значения, размерности, цвета, списки, переключатели и диапазоны.</li>
  <li>Добавлены массовые операции с категориями, выбор всех найденных товаров и выбор текущей страницы на мобильных устройствах.</li>
  <li>Улучшены адаптивный интерфейс, переключение операций и локализация на русском и английском языках.</li>
  <li>Исправлены применение остатков для выбранного склада и отображение пункта плагина в меню админ-панели Shop-Script.</li>
</ul>
```

## 1.2.0 — 2026-07-19

```html
<p><strong>Версия 1.2.0 — 19 июля 2026</strong></p>
<ul>
  <li>Добавлен фильтр типа складского учёта для массового изменения остатков без выбора конкретного склада.</li>
  <li>Добавлено массовое редактирование общих характеристик с множественными значениями: замена набора, добавление, удаление выбранных значений и полная очистка.</li>
  <li>Добавлена массовая установка или очистка видео с проверкой ссылок Rutube, VK, YouTube и Vimeo.</li>
  <li>Добавлено отдельное окно выполнения и результата операции; после успеха форма сбрасывается, после ошибки сохраняется для исправления.</li>
  <li>Уточнены русские и английские названия групп и операций.</li>
  <li>Добавлен защищённый откат последней успешной операции текущего пользователя в течение трёх часов с проверкой прав и конфликтов данных.</li>
  <li>Обновление установленного плагина создаёт журнал и закрытые rollback-таблицы повторно безопасными meta-update Webasyst.</li>
</ul>
```
