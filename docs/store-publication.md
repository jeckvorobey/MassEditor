# Материалы для публикации плагина «Массовый редактор» в Webasyst Store

## Позиционирование

`Массовый редактор` — backend-only плагин для Shop-Script уровня `$69`.

Плагин предназначен только для работы в админке и не изменяет витрину, корзину, оформление заказа или тему магазина.

## Краткое описание

Безопасное массовое редактирование товаров в Shop-Script: цены, compare price, остатки, видимость, доступность, базовые характеристики, категории, описание, теги и URL. Интерфейс доступен на русском и английском языках.

## Описание для карточки

`Массовый редактор` помогает быстро вносить одинаковые изменения сразу в несколько товаров прямо из backend Shop-Script.

Версия `1.1.0` поддерживает:

- массовое изменение цены;
- массовое изменение compare price;
- массовое изменение остатков с поддержкой складского учета;
- массовое изменение видимости товара;
- массовое изменение доступности товара;
- базовое редактирование характеристик товара;
- массовое изменение категорий;
- массовое обновление описания;
- массовую работу с тегами;
- массовое обновление URL товаров;
- выбор всех товаров по текущему фильтру;
- журнал последних операций;
- русский и английский интерфейс;
- настройку языка интерфейса: Auto, русский или English.

Плагин работает только в backend Shop-Script и ориентирован на повседневные задачи контент-менеджера или администратора каталога. Все изменения применяются только после подтверждения и пишутся в журнал.

## Ключевые преимущества

- работа внутри backend Shop-Script;
- массовые действия без изменения витрины;
- фильтрация и выбор товаров перед применением;
- выбор всех найденных товаров с серверной проверкой лимита;
- журнал выполненных операций;
- поддержка современного backend-интерфейса Webasyst;
- ручное переключение языка интерфейса и автоматический выбор языка по текущей локали Webasyst.

## Чеклист публикации версии 1.1.0

- Проверить `plugin.php`: версия `1.1.0`.
- Проверить Store price: `$69`.
- Запустить `bash tests/run-php-tests.sh`.
- Запустить `bash tests/run-js-tests.sh`.
- Запустить `php -l` для измененных PHP-файлов.
- Пересобрать `.mo` из `.po`, если публикационный архив включает бинарные каталоги переводов.
- На Docker-стенде вручную проверить операции `stock`, `features`, `categories` и выбор всех товаров по фильтру.
- Проверить RU и EN интерфейс.
- Подготовить новые скриншоты из реального backend-экрана.

## Что не обещать в карточке

Не указывать в описании функции, которых сейчас нет или которые не подтверждены текущим кодом:

- расширенное редактирование характеристик;
- SKU-характеристики и характеристики модификаций;
- множественные значения характеристик, цвета, диапазоны и размерности;
- создание новых значений характеристик;
- массовую работу с изображениями;
- массовую работу с видео;
- cross-selling и similar products;
- импорт и экспорт;
- автоматизацию по расписанию;
- работу с заказами;
- прямое определение страны или источника установки из Webasyst Store. В текущей версии Auto-режим языка использует локаль Webasyst.

## Формулировка про backend-only

Рекомендуемая фраза для карточки:

> Плагин работает только в backend Shop-Script и не влияет на витрину магазина.

## Рекомендуемые скриншоты

Нужно загружать только реальные экраны текущей версии плагина после входа в backend.

Рекомендуемый набор:

1. Общий экран раздела `Товары` с фильтрами, библиотекой операций и таблицей товаров.
2. Экран выбора операции изменения цены с заполненными параметрами.
3. Экран операции остатков.
4. Экран операции базового редактирования характеристик.
5. Экран операции категорий.
6. Модальное окно подтверждения для выбора всех товаров по фильтру.
7. Экран раздела `Журнал` с записями после тестового применения новых операций.
8. Экран `Настройки` с доступными параметрами плагина.
9. Экран `Настройки` после переключения языка на English.

## Подписи к скриншотам

1. Выбор товаров и массовых операций.
2. Настройка параметров изменения цены.
3. Управление остатками.
4. Базовое редактирование характеристик товара.
5. Управление категориями товаров.
6. Подтверждение изменений для всех товаров по фильтру.
7. Журнал последних массовых операций.
8. Настройки поведения плагина.
9. Выбор языка интерфейса.

## Ручная проверка цены `$69`

Перед публикацией вручную проверить цену в Webasyst Store:

1. Открыть карточку плагина в кабинете разработчика.
2. Убедиться, что цена указана `$69`.
3. Проверить, что описание не содержит старое позиционирование `$49`.
4. Проверить, что новые функции перечислены без обещания расширенных характеристик.

## Статус подготовки скриншотов

Автоматическая проверка может открыть страницу логина Webasyst, но без учетных данных нельзя снять реальные backend-экраны самого плагина.

Перед публикацией нужно:

1. Войти в локальный backend.
2. Открыть `Shop-Script -> Массовый редактор`.
3. Снять 9 экранов из списка выше.
4. Проверить, что на скриншотах нет чужих тестовых данных, технических ошибок и экрана логина.
5. Для двуязычной публикации дополнительно проверить экран настроек и основные вкладки после выбора `English`.
