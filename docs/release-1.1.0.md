# Массовый редактор 1.1.0

## Кратко

Версия `1.1.0` поднимает «Массовый редактор» до релиза уровня `$69`: кроме уже существующих операций с ценами, видимостью, доступностью, описаниями, тегами и URL добавлены операции для управления остатками, характеристиками, категориями и выбором товаров по фильтру или текущей странице.

Плагин остается backend-only и не меняет storefront, корзину, оформление заказа или тему магазина.

## Что добавлено

- Остатки: изменить обычный остаток без склада или выбрать склад, установить остаток, увеличить, уменьшить или сделать остаток бесконечным.
- Фильтр склада: отобрать товары по складу и видеть в списке остаток выбранного склада; режим «Все склады» сохраняет общую детализацию.
- Редактирование общих характеристик: строки, текст, числа, логические значения, размерности, цвета, списки, переключатели и диапазоны.
- Категории: добавить товар в категорию, удалить связь с категорией или заменить основную категорию.
- Выбор всех товаров по текущему фильтру: сервер заново строит выборку и применяет `operation_limit`.
- Выбор всех товаров текущей страницы на мобильных устройствах.
- Новые записи журнала для операций `stock`, `features` и `categories`.
- Новые RU/EN строки интерфейса и клиентской валидации.

## Что улучшено

- Библиотека операций теперь показывает больше реальных операций каталога, а не будущие заглушки.
- Модальное подтверждение учитывает новые режимы и значения.
- Серверная нормализация payload проверяет новые режимы, ID склада, ID категории, ID характеристики и лимит операции.
- Интерфейс стал удобнее на мобильных устройствах, а при смене операции значения предыдущей формы сбрасываются.
- Описание для Webasyst Store обновлено под позиционирование `$69`.

## Что исправлено

- Исправлено применение остатка для выбранного склада и пересчет итогового остатка товара.
- При фильтрации по складу таблица больше не подменяет его остаток общим остатком товара.
- Исправлено отображение пункта плагина в меню админ-панели Shop-Script.
- Исправлена языковая целостность уведомлений и сообщений валидации для `ru_RU` и `en_US`.

## Что не входит в эту версию

- SKU-характеристики и характеристики модификаций.
- Дочерние характеристики.
- Множественные значения характеристик.
- Создание новых значений характеристик.
- Массовая работа с фотографиями, видео, cross-selling, similar products и страницами товаров.
- Изменения storefront, корзины, заказов или темы магазина.

## Почему версия 1.1.0

Переход с `1.0.0` на `1.1.0` выбран по схеме `MAJOR.MINOR.PATCH`:

- `MAJOR` не меняется, потому что релиз обратно совместим и не ломает существующие сценарии.
- `MINOR` увеличивается, потому что добавлены новые возможности.
- `PATCH` остается `0`, потому что это не только исправление ошибок, а функциональный релиз.
